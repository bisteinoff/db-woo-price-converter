The plugin is used for converting the prices from one currency to another.

== Description ==

DB Woocommerce Price Converter is an easy solution to manage the prices if you need to convert them from one currency to another.

== Installation ==

1. Upload db-tagcloud folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
